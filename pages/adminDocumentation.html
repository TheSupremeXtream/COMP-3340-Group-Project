Administrator Documentation

This page explains how an administrator can manage the website using the admin interface. The admin area includes tools for changing the site template, editing product records, and managing user accounts.

  1. Log in as an administrator. Use an admin account from the Login page. After logging in, use the Admin link or open the admin pages directly.
  2. Switch site templates. Go to the Templates page to activate one of the three available site-wide templates: Light, Dark, or Holiday.
  3. Edit products. Go to the Products page to search existing items, edit a product, or add a new one. Each product can include category, title, brand, description, base price, stock, image filename, featured status, active status, and product options.
  4. Manage user accounts. Go to the Users page to search registered users and enable or disable customer accounts. Admin accounts are protected and cannot be disabled from this page.
  5. Use the monitoring page. Open the Monitor page to review service status, recent checks, and overall system health.
  6. Keep content easy to update. When adding or editing products, enter the image filename that matches a real image in the backend images folder. This makes the catalogue easier to maintain for future updates.
